# How to Run:

Minimum required node version: 13.0.0

Run `npm install` to install dependencies

Compile code:
`npm run build`

Run compiled code (prod version)
`npm run start`

Run dev mode:
`npm run dev`

Run tests:
`npm run test`

Routes:
* Report web view: `http://localhost:5000/`
* Report API: `http://localhost:5000/api`
